## Persona 1: Практичный Павел (Арендатор)

### Демография

| Параметр | Значение |
|----------|----------|
| **Возраст** | 32 года |
| **Пол** | Мужчина |
| **Семейное положение** | Женат, 1 ребёнок (5 лет) |
| **Образование** | Высшее (IT) |
| **Профессия** | Менеджер проектов |
| **Доход** | 180 000 ₽/мес (семейный) |
| **Жильё** | Новостройка, ЖК "Солнечный", 2-комн., ипотека |
| **Город** | Москва |

### Психография

**Ценности:**
- Рациональность и экономия ресурсов
- Качество жизни без лишних трат
- Время как главная валюта
- Минимализм: "зачем владеть, если можно использовать"

**Мотивации:**
- Решить задачу быстро и недорого
- Не захламлять квартиру редко используемыми вещами
- Быть хорошим семьянином (сделать полку, организовать праздник)

**Страхи:**
- Потратить деньги на вещь, которая будет пылиться
- Нарваться на мошенника при аренде у незнакомца
- Потратить выходные на поездку в прокат

### Поведение

**Технологии:**
- iPhone 14, MacBook
- Telegram — основной мессенджер
- Использует Яндекс.Такси, Самокат, Ozon
- Не любит устанавливать новые приложения

**Типичный сценарий:**
```
Суббота, 10:00
Павел решил повесить картину. Нужен перфоратор.

Раньше:
1. Пишет в домовой чат "У кого есть перфоратор?"
2. Ждёт ответа 2 часа
3. Пишет соседу, договаривается
4. Идёт за инструментом
5. Использует, возвращает
6. Стрессует, что может сломать

С Арендо-Соседи:
1. Открывает WebApp из домового чата
2. Находит перфоратор за 500₽/день
3. Бронирует, платит (включая депозит)
4. Спускается на 3 этажа, забирает
5. Использует, возвращает
6. Депозит возвращается автоматически
```

### Jobs to Be Done (JTBD)

| Job | Ситуация | Желаемый результат |
|-----|----------|-------------------|
| **Functional** | Нужен инструмент на 1 раз | Получить инструмент быстро и дёшево |
| **Emotional** | Хочет быть хорошим мужем/отцом | Сделать дело, не тратя семейный бюджет |
| **Social** | Не хочет выглядеть жадным | Получить вещь "официально", а не клянчить |

### Барьеры к использованию

| Барьер | Решение |
|--------|---------|
| Не доверяет незнакомцам | Это сосед, рейтинги видны |
| Боится повредить вещь | Страхование включено |
| Лень разбираться | UX максимально простой |
| Нет приложения | Telegram WebApp, ничего ставить не надо |

### Цитаты

> "Зачем мне перфоратор за 8000₽, если я использую его раз в год?"

> "В домовом чате спрашивать неудобно — как будто побираюсь"

> "Если у соседа взять и что-то сломается — как расплачиваться?"

### Метрики поведения (ожидаемые)

| Метрика | Значение |
|---------|----------|

| Средний чек | 800 ₽ |
| Категории | Инструменты (60%), Спорт (25%), Другое (15%) |
| Retention D30 | 35% |

## Persona 2: Заботливая Зина (Владелец)

### Демография

| Параметр | Значение |
|----------|----------|
| **Возраст** | 45 лет |
| **Пол** | Женщина |
| **Семейное положение** | Замужем, 2 детей (18 и 14 лет) |
| **Образование** | Высшее (бухгалтерия) |
| **Профессия** | Бухгалтер на удалёнке |
| **Доход** | 140 000 ₽/мес (семейный) |
| **Жильё** | Новостройка, ЖК "Солнечный", 3-комн., своя |
| **Город** | Москва |

### Психография

**Ценности:**
- Порядок и рачительность
- "Вещь должна работать, а не лежать"
- Добрые отношения с соседями
- Дополнительный доход не помешает

**Мотивации:**
- Заработать на том, что простаивает
- Помочь соседям (приятно быть полезной)
- Освободить место в квартире
- Социализация в доме

**Страхи:**
- Вещь повредят или не вернут
- Сложно будет разбираться с деньгами
- Соседи будут надоедать
- Придётся много времени тратить

### Поведение

**Технологии:**
- Samsung Galaxy, иногда iPad
- Telegram, WhatsApp, Одноклассники
- Использует Wildberries, Ozon
- Технически средне продвинута

**Типичный сценарий:**
```
У Зины в кладовке:
- Велосипед (сын вырос)
- Палатка (давно не ездили)
- Электросамокат (муж не катается)
- Детская коляска (внуков пока нет)
- Шуруповёрт (ремонт закончили)

Раньше:
1. Вещи лежат годами
2. Иногда одалживает соседям бесплатно
3. Переживает за сохранность
4. Думает продать, но жалко

С Арендо-Соседи:
1. Фотографирует вещи, выкладывает
2. Ставит комфортную цену (не завышает)
3. Получает заявки, подтверждает
4. Отдаёт вещь, получает деньги
5. Вещь возвращается, всё застраховано
6. Зарабатывает 3-5К в месяц
```

### Jobs to Be Done (JTBD)

| Job | Ситуация | Желаемый результат |
|-----|----------|-------------------|
| **Functional** | Вещи простаивают | Получать доход с простаивающего |
| **Emotional** | Хочет быть полезной | Помогать соседям, чувствовать значимость |
| **Social** | Хочет знать соседей | Расширить социальные связи в доме |

### Барьеры к использованию

| Барьер | Решение |
|--------|---------|
| Страх порчи вещи | Депозит + страхование |
| Сложно фотографировать | Простая форма, подсказки |
| Не знает, какую цену ставить | Рекомендация цены по категории |
| Боится назойливых арендаторов | Можно скрыть/отклонить заявку |

### Цитаты

> "Велосипед 3 года стоит, а выбросить жалко — хороший же!"

> "Соседка просила коляску, дала бесплатно. А если сломают — как спросить?"

> "Продавать на Авито замучаешься, а так пусть работает"

|---------|----------|

| Активность листингов | 70% |
| Доход в месяц | 3 000-5 000 ₽ |

## Persona 3: Молодая Марина (Арендатор активный)

### Демография

| Параметр | Значение |
|----------|----------|
| **Возраст** | 26 лет |
| **Пол** | Женщина |
| **Семейное положение** | Не замужем, живёт с парнем |
| **Образование** | Высшее (маркетинг) |
| **Профессия** | SMM-менеджер |
| **Доход** | 90 000 ₽/мес |
| **Жильё** | Аренда квартиры в новостройке |
| **Город** | Москва |

### Психография

**Ценности:**
- "Опыт важнее владения"
- Sustainability, осознанное потребление
- Активный образ жизни
- Социальные связи

**Мотивации:**
- Попробовать разное, не покупая
- Организовать вечеринку/пикник
- Быть экологичной
- Сэкономить на хобби

**Страхи:**
- Выглядеть "бедной" (просящей)
- Получить некачественную вещь
- Сложный процесс возврата

### Типичные сценарии использования

```
Сценарий 1: День рождения дома
- Нужен проектор для кино-вечеринки
- Колонка Bluetooth
- Гирлянды (если есть)
→ Аренда на 1 день: 1500₽ vs покупка 15000₽

Сценарий 2: Выходные на природе
- Палатка
- Мангал
- Спальники
→ Аренда на 2 дня: 2000₽ vs покупка 20000₽

Сценарий 3: Попробовать хобби
- SUP-доска
- Ролики
- Электросамокат
→ Аренда на день, чтобы понять — моё или нет
```

### Метрики поведения (ожидаемые)

| Метрика | Значение |
|---------|----------|
| Частота аренды | 2 раза в месяц |
| Средний чек | 1 200 ₽ |
| Категории | Мероприятия (40%), Спорт (35%), Техника (25%) |
| Repeat rate | 60% |

## Persona 5: Управдом Уваров (B2B — УК/ТСЖ)

### Демография

| Параметр | Значение |
|----------|----------|
| **Возраст** | 52 года |
| **Пол** | Мужчина |
| **Должность** | Председатель ТСЖ / Управляющий УК |
| **ЖК** | 500 квартир, 1200 жителей |

### Психография

**Ценности:**
- Порядок в доме
- Удовлетворённость жителей
- Инновации, но без рисков
- Отчётность и прозрачность

**Мотивации:**
- Повысить лояльность жителей
- Показать инновационность УК
- Дополнительный сервис без затрат
- Снизить конфликты между соседями

### Сценарий использования

```
Интеграция с УК:
1. Арендо-Соседи как официальный партнёр ЖК
2. Бот добавляется в официальный чат дома
3. УК продвигает сервис среди жителей
4. Опционально: УК предоставляет общие инструменты (мойка, пылесос для авто)
5. Отчётность для УК по активности

Выгода для УК:
- Повышение NPS жителей
- Снижение конфликтов
- Дополнительный сервис без CAPEX
- Возможная комиссия от сделок
```

### Метрики успеха

| Метрика | Цель |
|---------|------|
| Adoption rate в ЖК | >20% жителей |
| NPS жителей | +10 пунктов |
| Конфликты по вещам | -50% |

## Персона Anti-Patterns (кого НЕ таргетируем)

### 1. Профессиональный прокатчик
- Хочет сдавать 50+ вещей
- Требует B2B функционал
- **Почему нет:** Размывает соседское доверие, MVP не для этого

### 2. Перекупщик
- Берёт в аренду, чтобы субарендовать
- **Почему нет:** Мошенничество, конфликты

### 3. Скептик "мне ничего не надо"
- Никогда не арендует, не сдаёт
- **Почему нет:** Нет потребности, не конвертируется

### 4. Любитель халявы
- Хочет бесплатно или за копейки
- Торгуется до нуля
- **Почему нет:** Низкий LTV, конфликты с владельцами
