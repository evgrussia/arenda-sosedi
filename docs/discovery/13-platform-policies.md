# Политики и правила платформы Арендо-Соседи

## 1. Общие положения

### 1.1 Назначение документа

Данный документ определяет правила использования платформы Арендо-Соседи, обязанности участников и меры ответственности за нарушения. Правила обязательны для всех пользователей.

### 1.2 Определения

| Термин | Определение |
|--------|-------------|
| **Платформа** | Сервис Арендо-Соседи (Telegram WebApp) |
| **Владелец** | Пользователь, размещающий вещи для аренды |
| **Арендатор** | Пользователь, берущий вещи в аренду |
| **Транзакция** | Процесс аренды от бронирования до возврата |
| **Модерация** | Проверка контента и действий администрацией |

### 1.3 Принятие правил

- Регистрация на платформе означает согласие с правилами
- Незнание правил не освобождает от ответственности
- Правила могут обновляться с уведомлением пользователей

## 2. Правила для пользователей

### 2.1 Регистрация и верификация

#### 2.1.1 Требования к регистрации

| Требование | Обязательность | Описание |
|------------|----------------|----------|
| Telegram-аккаунт | Обязательно | Действующий аккаунт |
| Возраст 18+ | Обязательно | Совершеннолетие |
| Реальное имя | Обязательно | Для доверия соседей |
| Фото профиля | Рекомендуется | Реальное фото |
| Привязка к ЖК | Обязательно | Подтверждение проживания |

#### 2.1.2 Верификация адреса

**Допустимые способы:**
1. Геолокация при регистрации (±500м от ЖК)
2. Приглашение от верифицированного соседа
3. Документ от УК/ТСЖ (manual review)

**Запрещено:**
- Указывать чужой адрес
- Использовать VPN/фейковую геолокацию
- Создавать дублирующие аккаунты

### 2.2 Поведение на платформе

#### 2.2.1 Кодекс поведения

**Ожидаемое поведение:**
- Вежливое общение в чате
- Своевременные ответы (в течение 24ч)
- Честное описание состояния вещей
- Соблюдение договорённостей
- Уважение к имуществу соседей

**Недопустимое поведение:**
- Оскорбления, угрозы, дискриминация
- Spam, реклама сторонних сервисов
- Мошенничество и обман
- Передача вещей третьим лицам
- Использование вещей не по назначению

#### 2.2.2 Ответственность

| Нарушение | Санкция |
|-----------|---------|
| Первое предупреждение | Уведомление |
| Повторное нарушение | Временная блокировка (7 дней) |
| Систематические нарушения | Блокировка на 30 дней |
| Грубые нарушения | Перманентная блокировка |
| Мошенничество | Блокировка + передача в правоохранительные органы |

## 3. Правила размещения объявлений

### 3.1 Допустимые категории

| Категория | Примеры | Особые требования |
|-----------|---------|-------------------|
| **Инструменты** | Дрель, шуруповёрт, лобзик | Проверка работоспособности |
| **Спорт/туризм** | Велосипед, палатка, лыжи | Указание размеров |
| **Электроника** | Проектор, колонки, камера | Комплектность |
| **Детские товары** | Коляска, автокресло, манеж | Сертификаты безопасности |
| **Event-товары** | Мангал, шатёр, настолки | Указание вместимости |
| **Прочее** | Чемодан, пылесос, посуда | — |

### 3.2 Запрещённые товары

**Абсолютный запрет:**
- Оружие и боеприпасы
- Наркотические вещества
- Контрафакт и пиратская продукция
- Документы и ценные бумаги
- Животные
- Продукты питания
- Медицинские препараты
- Товары для взрослых (18+)
- Взрывчатые/горючие вещества

**Ограничения:**
- Автомобили — только через партнёров (будущее)
- Недвижимость — запрещено
- Одежда — только event (костюмы)
- Ювелирные изделия — запрещено

### 3.3 Требования к объявлениям

#### 3.3.1 Обязательные элементы

```yaml
Название:
  - Длина: 5-100 символов
  - Формат: "[Категория] Название модели"
  - Пример: "Дрель Bosch GSB 13RE"

Описание:
  - Длина: 20-2000 символов
  - Содержание: состояние, комплектность, особенности
  - Честность: указывать дефекты

Фотографии:
  - Минимум: 1 фото
  - Максимум: 10 фото
  - Формат: JPG, PNG
  - Качество: разрешение от 800×600
  - Реальные фото: не стоковые

Цена:
  - Указание за сутки
  - Минимум: 50 ₽
  - Максимум: 50 000 ₽
  - Валюта: только рубли

Залог:
  - Опционально
  - Обоснованный размер
  - Максимум: рыночная стоимость × 0,8
```

#### 3.3.2 Модерация объявлений

| Этап | Время | Критерии |
|------|-------|----------|
| Авто-проверка | Мгновенно | Spam, запрещёнка, формат |
| AI-модерация | 1-5 мин | Качество фото, описание |
| Manual review | До 24ч | Спорные случаи |

**Причины отклонения:**
- Недостаточное описание
- Плохое качество фото
- Неадекватная цена
- Запрещённый товар
- Дубликат объявления

## 4. Правила транзакций

### 4.1 Процесс бронирования

#### 4.1.1 Обязанности арендатора

1. **До бронирования:**
   - Изучить описание и фото
   - Задать уточняющие вопросы
   - Убедиться в наличии дат

2. **При бронировании:**
   - Указать корректные даты
   - Оплатить аренду + залог (если требуется)
   - Подтвердить готовность забрать вещь

3. **После подтверждения:**
   - Явиться в согласованное время (±15 мин)
   - Проверить вещь при получении
   - Подписать акт приёма (в приложении)

#### 4.1.2 Обязанности владельца

1. **До бронирования:**
   - Поддерживать актуальность календаря
   - Отвечать на запросы в течение 24ч
   - Честно описывать состояние вещи

2. **При бронировании:**
   - Подтвердить/отклонить в течение 24ч
   - Согласовать время/место передачи

3. **При передаче:**
   - Предоставить вещь в заявленном состоянии
   - Провести краткий инструктаж
   - Подписать акт передачи (в приложении)

### 4.2 Правила аренды

#### 4.2.1 Сроки аренды

| Тип | Минимум | Максимум |
|-----|---------|----------|
| Стандартная | 1 день | 30 дней |
| Продлённая | По согласованию | 90 дней |

#### 4.2.2 Использование вещей

**Арендатор обязан:**
- Использовать вещь по назначению
- Соблюдать инструкции владельца
- Не передавать третьим лицам
- Не вывозить за пределы города
- Сообщать о поломках немедленно

**Запрещено:**
- Разбирать/модифицировать вещь
- Использовать в коммерческих целях
- Сдавать в субаренду
- Использовать в опасных условиях

### 4.3 Возврат

#### 4.3.1 Процедура возврата

1. Арендатор уведомляет о готовности вернуть
2. Согласование времени/места
3. Совместный осмотр вещи
4. Подписание акта возврата
5. Возврат залога (в течение 24ч)

#### 4.3.2 Состояние при возврате

| Состояние | Действие |
|-----------|----------|
| Без изменений | Полный возврат залога |
| Нормальный износ | Полный возврат залога |
| Мелкие повреждения | Частичное удержание залога |
| Серьёзные повреждения | Удержание залога + компенсация |
| Утрата вещи | Компенсация рыночной стоимости |

## 5. Финансовые правила

### 5.1 Ценообразование

#### 5.1.1 Рекомендации по ценам

| Категория | Рекомендуемая цена/день |
|-----------|-------------------------|
| Инструменты | 3-5% от стоимости |
| Электроника | 5-7% от стоимости |
| Спортинвентарь | 3-5% от стоимости |
| Детские товары | 2-4% от стоимости |
| Event-товары | 5-10% от стоимости |

#### 5.1.2 Запрещённые практики

- Демпинг (цена < себестоимости услуги)
- Скрытые платежи
- Навязывание дополнительных услуг
- Оплата вне платформы (обход комиссии)

### 5.2 Комиссия платформы

```yaml
Базовая комиссия: 12,5%
Взимается с: Арендатора
Включает:
  - Обработка платежа
  - Страховое покрытие (базовое)
  - Поддержка 24/7
  - Разрешение споров

Скидки:
  - Премиум-владельцы: 10%
  - Высокий объём: индивидуально
```

### 5.3 Залог (депозит)

#### 5.3.1 Правила залога

| Параметр | Правило |
|----------|---------|
| Обязательность | На усмотрение владельца |
| Максимум | 80% рыночной стоимости |
| Хранение | Эскроу на платформе |
| Возврат | В течение 24ч после акта возврата |

#### 5.3.2 Удержание залога

**Допустимые причины:**
- Повреждение вещи (с фото-доказательствами)
- Просрочка возврата (штраф 20%/день)
- Утрата вещи
- Недопустимое использование

**Процедура:**
1. Владелец инициирует претензию
2. Арендатор получает уведомление
3. 48 часов на возражения
4. Модератор принимает решение
5. Автоматическое удержание/возврат

### 5.4 Отмена и возвраты

#### 5.4.1 Отмена арендатором

| Срок до начала аренды | Возврат |
|-----------------------|---------|
| >7 дней | 100% |
| 3-7 дней | 75% |
| 1-3 дня | 50% |
| <24 часов | 0% |

#### 5.4.2 Отмена владельцем

| Срок до начала аренды | Последствия |
|-----------------------|-------------|
| Любой срок | 100% возврат арендатору |
| <24 часов | Штраф 15% от суммы аренды |
| Систематические отмены | Снижение рейтинга + warning |

## 6. Система рейтингов и отзывов

### 6.1 Структура рейтинга

```yaml
Общий рейтинг: 1-5 звёзд

Компоненты для владельца:
  - Соответствие описанию: 25%
  - Состояние вещи: 25%
  - Коммуникация: 25%
  - Пунктуальность: 25%

Компоненты для арендатора:
  - Бережное отношение: 30%
  - Пунктуальность: 30%
  - Коммуникация: 20%
  - Соблюдение правил: 20%
```

### 6.2 Правила отзывов

#### 6.2.1 Допустимый контент

- Честная оценка опыта
- Конструктивная критика
- Факты и наблюдения
- Рекомендации другим пользователям

#### 6.2.2 Запрещённый контент

- Оскорбления и мат
- Ложная информация
- Персональные данные
- Реклама и spam
- Угрозы и шантаж
- Дискриминация

#### 6.2.3 Модерация отзывов

| Действие | Время |
|----------|-------|
| Авто-фильтр | Мгновенно |
| Жалоба пользователя | Рассмотрение 24-48ч |
| Удаление | С уведомлением автора |
| Апелляция | 7 дней на подачу |

### 6.3 Влияние рейтинга

| Рейтинг | Статус | Эффекты |
|---------|--------|---------|
| 4.5-5.0 | Отличный | Повышенная видимость, бейдж |
| 4.0-4.4 | Хороший | Стандартная видимость |
| 3.5-3.9 | Удовлетворительный | Предупреждение |
| 3.0-3.4 | Низкий | Ограничение функций |
| <3.0 | Критический | Блокировка до апелляции |

## 7. Разрешение споров

### 7.1 Виды споров

| Тип спора | Примеры |
|-----------|---------|
| Состояние вещи | Не соответствует описанию |
| Повреждения | Спор о причинённом ущербе |
| Финансовый | Спор о сумме компенсации |
| Поведенческий | Нарушение договорённостей |
| Безопасность | Угрозы, мошенничество |

### 7.2 Процедура разрешения

```
Шаг 1: Прямые переговоры (24-48ч)
  ↓ (не решено)
Шаг 2: Медиация платформы (до 5 дней)
  ↓ (не решено)
Шаг 3: Арбитраж платформы (решение обязательно)
  ↓ (несогласие)
Шаг 4: Внешние инстанции (суд, Роспотребнадзор)
```

### 7.3 Правила арбитража

#### 7.3.1 Принципы

- Презумпция добросовестности
- Решение на основе доказательств
- Равное отношение к сторонам
- Прозрачность процесса

#### 7.3.2 Доказательства

**Принимаются:**
- Фото/видео с датой
- Переписка в чате платформы
- Акты приёма-передачи
- Показания свидетелей (других пользователей)

**Не принимаются:**
- Устные заявления без подтверждения
- Переписка вне платформы
- Материалы сомнительного происхождения

### 7.4 Компенсации

| Ситуация | Компенсация |
|----------|-------------|
| Вещь не соответствует описанию | 100% возврат + 10% бонус |
| Отмена владельцем в последний момент | 100% возврат + 15% компенсация |
| Повреждение вещи арендатором | Из залога + доплата |
| Утеря вещи | Рыночная стоимость |
| Мошенничество | Полная компенсация + блокировка виновного |

## 8. Безопасность и конфиденциальность

### 8.1 Защита данных

#### 8.1.1 Собираемые данные

| Данные | Цель | Хранение |
|--------|------|----------|
| Telegram ID | Идентификация | Постоянно |
| Имя, фото | Профиль | До удаления аккаунта |
| Геолокация | Привязка к ЖК | Однократно |
| История транзакций | Сервис | 3 года |
| Переписка | Споры | 1 год |
| Платёжные данные | YooKassa | Не храним |

#### 8.1.2 Права пользователей

- Доступ к своим данным
- Исправление неточностей
- Удаление аккаунта (с историей)
- Выгрузка данных (GDPR-like)
- Отзыв согласий

### 8.2 Безопасность транзакций

#### 8.2.1 Меры защиты

```yaml
Платежи:
  - PCI DSS compliant (YooKassa)
  - Эскроу для залогов
  - Без хранения карт на платформе

Аккаунты:
  - Telegram-авторизация (2FA)
  - Верификация адреса
  - Лимиты на новые аккаунты

Коммуникации:
  - Чат только внутри платформы
  - Модерация сообщений
  - Блокировка контактов вне платформы
```

#### 8.2.2 Рекомендации пользователям

**Рекомендуем:**
- Общаться только в чате платформы
- Проверять вещь при получении
- Фотографировать до/после
- Не передавать деньги напрямую

**Не рекомендуем:**
- Делиться номером телефона
- Встречаться в небезопасных местах
- Оплачивать вне платформы
- Передавать вещи без акта

### 8.3 Противодействие мошенничеству

#### 8.3.1 Признаки мошенничества

- Просьба оплатить вне платформы
- Слишком низкая цена
- Давление на срочность
- Отказ от осмотра
- Новый аккаунт без отзывов + дорогой товар

#### 8.3.2 Действия при подозрении

1. Прекратить общение
2. Не совершать оплату
3. Сообщить в поддержку
4. Заблокировать пользователя

## 9. Модерация и enforcement

### 9.1 Типы нарушений

#### 9.1.1 Классификация

| Уровень | Примеры | Санкция |
|---------|---------|---------|
| **Незначительное** | Некорректное описание, поздний ответ | Warning |
| **Среднее** | Отмена без причины, грубость | Временная блокировка |
| **Серьёзное** | Обман о состоянии, no-show | Блокировка 30 дней |
| **Критическое** | Мошенничество, угрозы | Перманентный бан |

### 9.2 Процедура enforcement

```
1. Выявление нарушения
   ├─ Автоматическое (AI/rules)
   └─ Жалоба пользователя

2. Проверка
   ├─ Сбор доказательств
   └─ Анализ контекста

3. Решение
   ├─ Предупреждение
   ├─ Санкция
   └─ Отклонение жалобы

4. Уведомление
   ├─ Нарушитель
   └─ Заявитель

5. Апелляция (7 дней)
   └─ Пересмотр (если есть основания)
```

### 9.3 Апелляции

#### 9.3.1 Процедура

1. Подать апелляцию в течение 7 дней
2. Предоставить дополнительные доказательства
3. Ожидать рассмотрения (до 5 рабочих дней)
4. Получить финальное решение

#### 9.3.2 Основания для апелляции

- Новые доказательства
- Технические ошибки
- Несоразмерность санкции
- Ошибка в идентификации
